== Description ==

Display WP e-Commerce store sales details to site visitors through a WordPress Widget.

See http://www.visser.com.au/wp-ecommerce/plugins/sales-widget/

== Installation ==

1. Upload the folder 'wp-e-commerce-sales-widget' to the '/wp-content/plugins/' directory

2. Activate 'Sales Widget for WP e-Commerce' through the 'Plugins' menu in WordPress

== Usage ==

1. Open Appearance > Widgets

2. Assign the Sales Widget to a WordPress Sidebar

Done!
